# Knowledgebase 

This is a repo for extracting knowledge from the internet or from other local data sources to form-up prompts.

## How to use
uvicorn server:app --reload

## How to contribute
1. Fork the repo
2. Create a new branch
3. Make your changes
4. Create a pull request
